# Kushalayu 🏥

AI-powered multilingual health assistant built for rural India.

## Features
- 🏠 Home Dashboard
- 🚨 Emergency Services
- 👨‍👩‍👧 Family Risk Assessment
- ⏳ Health Time Machine
- 📋 Report Analysis
- 🎙️ Voice Assistant
- 💆 Wellness Tracker

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **AI:** Google Gemini, AWS Polly, AWS Textract

## Setup

```bash
# Client
cd client
npm install
npm run dev

# Server
cd server
npm install
node index.js
```
